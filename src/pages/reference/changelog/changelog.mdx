# **Changelog**

This page documents notable changes to the lomi. API and related services.

## **Version 1 (v1)**

*   **Initial Release:** The first version of our REST API (`/v1/`).
*   **Core Resources:** Introduced foundational endpoints for managing:
    *   [Authentication](./../core/authentication)
    *   [Merchants](./../core/merchants)
    *   [Customers](./../core/customers)
    *   [Products](./../core/products)
    *   [Transactions](./../core/transactions)
    *   [Subscription Plans](./../core/subscriptions)
    *   [Providers](./../core/providers)
    *   [Webhooks](./../core/webhooks)
    *   [Checkout Sessions](./../core/checkout-sessions)
    *   [Customer Subscriptions](./../core/customer-subscriptions)
    *   [Payment Links](./../core/payment-links)
    *   [Refunds](./../core/refunds) (Beta)
*   **Environments:** Established separate Test and Live environments.
*   **Error Handling:** Standardized error response structure.
