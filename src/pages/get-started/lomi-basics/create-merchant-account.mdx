# Merchant Account

## Creating an Account

1. Visit [lomi.africa](https://lomi.africa/sign-up)
2. Provide business details:
   - Legal business name
   - Tax ID
   - Country of operation
   - Contact information

## Account Verification

Required documents:
- Business registration
- Owner or legal signatory Government ID
- Proof of address
- Bank account details

## API Access

Once verified, access your API credentials:

```typescript
// Test Environment
const testClient = new LomiSDK({
  apiKey: 'lomi_sk_test_...',
  baseUrl: 'https://sandbox.api.lomi.africa/v1'
});

// Production Environment
const liveClient = new LomiSDK({
  apiKey: 'lomi_sk_live_...',
  baseUrl: 'https://api.lomi.africa/v1'
});
```

## Merchant Information

Retrieve merchant details:
```typescript
const merchant = await lomi.merchants.get('merchant_id');
```

Response example:
```json
{
  "merchant_id": "mer_123",
  "business_name": "Example Corp",
  "country": "SN",
  "currency": "XOF",
  "status": "active",
  "capabilities": {
    "orange_money_payments": "active",
    "wave_payments": "active",
    "card_payments": "pending"
  }
}
```
