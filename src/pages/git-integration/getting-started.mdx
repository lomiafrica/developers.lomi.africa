# Getting Started

To start contributing to lomi., follow these steps:

1. Fork the lomi. repository on GitHub.

2. Clone your forked repository to your local machine.
   ```bash
   git clone https://github.com/lomiafrica/lomi.
   ```
3. Create a new branch for your feature, bug fix, or improvement.
   ```bash
   git checkout -b my-feature-branch
   ```
4. Make your changes and commit them with descriptive messages.
   ```bash
   git commit -m "Add new feature: my-feature-update"
   ```
5. Push your changes to your forked repository.
   ```bash
   git push origin my-feature-branch
   ```
6. Open a pull request on the main lomi. repository, describing your changes and referencing any relevant issues.

Before submitting a pull request, please ensure that your code follows our contributing guidelines and passes all tests.

If you have any questions or need assistance, feel free to reach out to our team at [hello@lomi.africa](mailto:hello@lomi.africa) or open an issue on GitHub. We plan to launch a Discord server soon for more interactive discussions!
