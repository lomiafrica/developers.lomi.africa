# Code of Conduct

## Our Pledge

We as members, contributors, and leaders pledge to make participation in our community a harassment-free experience for everyone, regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, caste, color, religion, or sexual identity and orientation.

## Our Standards

### Positive Behavior

- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community
- Showing empathy towards other community members

### Unacceptable Behavior

- The use of sexualized language or imagery
- Trolling, insulting/derogatory comments, and personal or political attacks
- Public or private harassment
- Publishing others' private information without explicit permission
- Other conduct which could reasonably be considered inappropriate

## Enforcement Responsibilities

Community leaders are responsible for clarifying and enforcing our standards of acceptable behavior and will take appropriate and fair corrective action in response to any behavior that they deem inappropriate, threatening, offensive, or harmful.

## Scope

This Code of Conduct applies within all community spaces, and also applies when an individual is officially representing the community in public spaces.

## Enforcement

### 1. Correction
```bash
# Private message
"Your comment [link] was inappropriate because..."
```

### 2. Warning
```bash
# Official warning
"This is a formal warning. Further violations will result in..."
```

### 3. Temporary Ban
```bash
# Temporary restriction
"Your access has been restricted for [duration] due to..."
```

### 4. Permanent Ban
```bash
# Permanent restriction
"Due to repeated violations, your access has been permanently revoked."
```

## Reporting Guidelines

### 1. Contact Information

- Email: conduct@lomi.africa
- GitHub: Open an issue with `[COC]` prefix
- Slack: Direct message to @moderators

### 2. Report Format

```markdown
## Incident Report
- Date/Time: [When it occurred]
- Location: [Where it occurred]
- Description: [What happened]
- Evidence: [Links/Screenshots]
- Witnesses: [If any]
```

### 3. Response Time

- Initial response: Within 24 hours
- Investigation: 1-7 days
- Resolution: As soon as investigation completes
