# lomi. CLI

The lomi. CLI helps you integrate lomi.'s payment services into your applications. It provides tools for:

- Project initialization and configuration
- Payment operations management
- Webhook endpoint configuration
- API key management
- Integration status monitoring

## Installation

Install the lomi. CLI using npm:

```bash copy
npm install lomi.cli
```

## Authentication

The CLI requires authentication to interact with the lomi. API. You can authenticate using:

1. **Environment Variable**:
```bash copy
export LOMI_API_KEY=your_api_key_here
```

2. **Interactive Login**:
```bash copy
lomi login
```

## Core Commands

### `lomi init`

Initializes a new lomi. project in your current directory:

```bash copy
lomi init
```

This command:
- Creates a `lib/lomi.ts` file with client configuration
- Sets up environment variables in `.env`
- Installs the lomi. client package
- Configures your API key

### `lomi dev`

Starts the development environment:

```bash copy
lomi dev
```

Features:
- Local development server
- Integration testing support
- Environment configuration

### `lomi status`

Displays your integration status:

```bash copy
lomi status
```

Shows:
- API connectivity status
- Account information
- Integration health

### `lomi payments`

Manage payment operations:

```bash copy
# List payments
lomi payments list

# Get payment details
lomi payments get <payment_id>
```

### `lomi webhook`

Manage webhook endpoints:

```bash copy
# Register new webhook
lomi webhook register

# List webhooks
lomi webhook list --merchant-id <merchant_id>
```

### `lomi api-key`

Manage API keys:

```bash copy
# List keys
lomi api-key list

# Create key
lomi api-key create
```

## Global Options

- `--api-key`: Set API key
- `--env`: Set environment (default: 'production')
- `--json`: JSON output format
